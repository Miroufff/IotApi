IotApi
=======================

A Symfony project created on October 26, 2016, 9:21 am.

## Features

* PHP 7.0
* Influx 1.0
* Composer 1.3
* Mysql 5.6

## Install

Clone the project:
```bash
git clone https://github.com/Miroufff/IotApi
```  

Install via composer:
```bash
composer update
```  

Start the server
```bash
php app/console server:start 0.0.0.0:8000
```
